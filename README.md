### Full Stack Developer | Laravel · NestJS · Angular · Vue.js · React Native

I'm a Full Stack Developer whose path into software started in IT support and network infrastructure — troubleshooting, user support, and setting up networks (including a full school network implementation). That hands-on, problem-first mindset still shapes how I approach development today.
